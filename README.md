# BookBot

BookBot is a Python command-line tool that analyzes a text file and prints:

- total word count
- character frequency counts (alphabetic characters only)

## Requirements

- Python 3

## Usage

```bash
python3 main.py <path_to_book>
```

Example:

```bash
python3 main.py books/frankenstein.txt
```

The script prints a summary report with overall word count and per-character counts sorted from highest to lowest.
